# Jeevana Life OS

## Project Rules

- Use Bahasa Indonesia for all in-app copywriting
- Maintain casual, friendly, Gen-Z tone
- AI Chat is the primary interface — all other modules are secondary
- Follow existing design system (ink, brand, grape, canvas colors)
- Use TanStack Router for routing, TanStack Query for data fetching
- All database access goes through Supabase with RLS
